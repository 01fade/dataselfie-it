  • How does Data Selfie work?

    When you install the Data Selfie browser extension, it will run in the background as you use Facebook on your desktop. Data Selfie will record your interactions on the platform. The extension will record: clicks on likes in your newsfeed, clicks on newsfeed links to external sites, duration spent on different posts and the specifics of those posts (authors, images and text) in your newsfeed, anything you type, and time spent on Facebook overall. The more you use Facebook the more data is recorded. You can look at the Data Selfie dashboard and get a sense of what Facebook could know about your personality based on this data. The extension shows predictive analytics and insights by using machine learning APIs: Apply Magic Sauce and IBM Watson. So, part of the tracked data needs to be processed on the server, in order to so it is sent without any indicators, like name or Facebook ID, that are tied to you to the Data Selfie server.

    Go to the privacy tab for more specific information about structure and process that affect your personal data.

  • Will you store my data somewhere?

    Your data is stored locally on your own computer. The Data Selfie server does not keep a data base of user data. It only passes the necessary data to the APIs - without your personal information such as name, gender, etc. The third-party services might log or store anonymized data or use the results to improve their services, but again they can't identify you as you, as no data about you specifically is sent along.

    Go to the privacy tab for more specific information about structure and process that affect your personal data.

  • What data does the extension send to the API?

    Whatever you type in the Facebook platform, the Like IDs of pages you have interacted with the most, the texts and images in the posts you have looked at will be send to the Data Selfie server and then directly (without storing that data in a data base) to third-party APIs. Your name, Facebook ID or profile link, are not sent anywhere.

    Go to the privacy tab for more specific information about structure and process that affect your personal data.

  • What third-party machine learning API services are you using?

    We use two machine learning API services:

     1. We are using the Apply Magic Sauce API that uses Facebook Like ID’s to predict a users psycho-demographic traits based on their digital footprints. You can learn more in the Apply Magic Sauce API documentation.

    Here is the white paper that is the basis for the API:

     2. We are using IBM Watson. Specifically we are using the Personality Insights and Alchemy API. We may use the Visual Recognition and Language Translation in further releases.

  • What will the third-party API Services do with my data?

    It is common for them to use anonymized data to improve their models to make better predictions in the future.

    For the IBM machine learning, we have "opted out" so that the requests from the Data Selfie server are not logged and not used for training their models. You can read more here: Controlling Request Logging for Watson Services

    Go to the privacy tab for more specific information about structure and process that affect your personal data.

  • How often will I see updated predictions?

    Data Selfie is a free tool that is funded by grants. In order to keep costs in check, we have to limit the daily API calls so that as many people as possible can use Data Selfie. We hope people will be nice and courteous to the community so that Data Selfie can be available for all. Updated predictions will depend on your Facebook usage and depend on whether or not there is significant change to your data that makes a new predictions plausible.
    If you really want to play with this more, refer to the next question.

  • What languages are supported?

    To analyze the content in the posts you look at Data Selfie only supports English, German, French, Italian, Portuguese, Russian, Spanish and Swedish. We use IBM's AlchemyLanguage service for that. Predictions based on the text you type, only English, Spanish, Japanese and Arabic are supported. We use IBM's Personality Insights service for that. This might change in the future.

  • Do you have any connections to Cambridge Analytica?

    There has been a wide spread discussion about how Cambridge Analytica might have used psychometrics (Big 5 or OCEAN) in their efforts in the Brexit referendum and the US Election 2016 (see articles from e.g. Motherboard and Das Magazin). We have no connection to this company and will never have. The University of Cambridge Psychometrics Centre has released a statement to clarify that they are in no way affiliated with Cambridge Analytica.

  • Are you using Facebook’s Graph API?

    Data Selfie is not using Facebook’s API. The extension looks at the rendered front-end (the DOM) of the user's Facebook page in the browser – the HTML elements and their content – and with JavaScript the browser extension can detect when you scroll, when you type and when you click something.

  • What permissions does the extension require?

    The extension will ask you for the following permissions: “Read and change your data on api.dataselfie.it and facebook.com” and “Manage your downloads”. The extension needs to be able to monitor activity on facebook.com and change some interface elements, so the user can receives feedback of what is tracked. api.dataselfie.it is the url to make requests for prediction; this permission is necessary in order to communicate with the Data Selfie server and APIs. “Manage your downloads” is for backing up your data into one single file on your computer.

  • What can I do with my tracked data?

    You can export your tracked data into a JSON file anywhere on your computer. You can import your tracked data from a JSON file with the same data structure. Finally you can delete your tracked data and the prediction results saved locally in your browser. Note, if you didn't export the data, it's impossible to retrieve that data after you deleted it as there is no record of it on the Data Selfie server.

  • Will you make a mobile version?

    We recognize that the vast majority of social media usage is done on a mobile device. From the beginning of this project we explored making a mobile version. However, there is no way for us to access Facebook’s native mobile application. (And that is a good thing!) Therefore, the only way to make a mobile version would be to launch our own browser as a native application and implement Data Selfie features. We feel that this approach is not in line with our mission of providing good user experience design. Thus, we have no intention of making a mobile version.

  • Is Data Selfie free?

    Data Selfie has been designed as a open source, free tool that aims to create awareness in society about the erosion of privacy and to encourage internet user to be critical thinkers.

  • Is Data Selfie open source?

    Yes! We believe in an open, free, and democratic internet. Our intention with this project is to be totally transparent. Data Selfie will never use your data for research purposes or for commercial purposes as user data is kept in a data base. You can check out our code at our GitHub account.

    Continous covering of API costs and maintenance efforts, and the development of new features are made possible by a Flash Grant from the Shuttleworth Foundation and a Media Grant from the Mozilla Foundation, supported [in part] by a grant from the Open Society Foundations.
